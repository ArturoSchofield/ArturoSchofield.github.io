Encargo 3

Animaciones digitales

Se decidio utilizar el programa externo presentado en la clase llamado “gaea” para generar superficies procedurales distintas donde probar diferentes enfoques de luz y sombra.

Dentro de esta experimentación se decidio introducir una rotación pero esta vez aplicando la sombra sobre una superficie deformante, logrando con esto generar un efecto diferente

Luego se intento generar un efecto de volumen sobre la superficie, dando la sensacion de modificación sobre la misma. Esto genera una ilusión que impide determinar si lo que cambia es la superficie o la sombra que se proyecta sobre esta.
