Luz y Sombra

En este encargo se busca que los alumnos entiendan como se difumina la luz por un material, del mismo modo que practicar el dibujo a mano alzada, por lo que tendrán que generar 3 elementos que interactuen con una fuente de luz para generar una composición, luego deberán generar 9 diferentes fracmentos o vistas de cada elementos, para finalmente dibujar estos fracmentos utilizando tecnicas de achurado.
