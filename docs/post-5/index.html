Encargo 5

En primer lugar debemos posicionar el modelo importado

Para esto utilizamos el comando move y dentro seleccionamos la opción components

Una vez posicionada la mano debes aplicar planos de corte, luego crearemos offset a partir de la visualizacion del corte

Generamos estos offset a la altura del codo, la muñeca y antes de los dedos

Teniendo las lineas realizaremos un loft para crear la superficie a trabajar

Con la herramienta modificar ajustaremos los puntos de intersección de la superficie, tambien añadiremos un agujero para el pulgar

Finalmente tendremos el modelo terminado, solo añadiremos un material que sera visible en la visualizacion de fusion

Invernadero

Generaremos un modelo a partir de una imagen. Para ello importaremos la imagen

Crearemos un diagrama con restricciones, por ejemplo fijaremos el ancho como medida estandar y todo el resto estara en función del ancho del invernadero

Crearemos un circulo con un diametro estandar de un tuvo que se puede comprar en cualquier ferreteria

Se levanta el modelo a partir de las medidas que hemos construido

Se repite nuestro modelo para crear los pilares del invernadero

Crearemos una superficie para cubrir el modelo

Se puede modificar los parametros que afectan a todo el modelo, dando la oportunidad de cambiar dependiendo de las cosas que se encuentren en la ferreteria

Plantación

Crearemos una plantación vertical a traves de una foto.

Crearemos un sketch con las formas basicas

Se extrulle el modelo a partir de las lineas

Se genera el pilar de la misma forma que la base

Se coloca un espiral que rodea el pilar

Se extrae la curva de la espiral y se posiciona un plano a traves de él, luego se crea una esfera en uno de los extremos

Se extriene un patron utilizando la curva del espiral

Se multiplica el modelo en 2 ejes para tener la plantación vertical
